Using Python and Flask

Do the following steps instead of the first step in the tutorial

  1. Create a folder on your desktop called "api"
  2. Open IDLE or the IDE of your choice and save a file called "app.py"
  3. Spotlight search for "Terminal" and open it
  4. Type in "cd Desktop/api" and hit Return. If you saved your python file in a different folder, replace Desktop/api with the path to that folder
  5. Type in "pip3 install flask flask-restful --user" and hit Return
  6. Do not update Python!
  7. Follow the tutorial or use the docs to begin making your api in the file app.py
  8. To run the program, type in "python app.py" in Terminal, hit return, and once you see the text "Running on <url>", copy paste it into a browser to see the output
  9. If you encounter any errors, ask one of us
